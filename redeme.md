# 数字信息实验室-研究工作台系统 

基于 Spring Boot (后端) 和 Vue 3 (前端) 的 B/S 架构系统，用于辅助研究生工作。
## 系统架构

- **后端**: Java 17, Spring Boot 3.2, Spring Data JPA, Spring Security + JWT
- **前端**: Vue 3, Vite, Element Plus, Pinia, Axios
- **数据库**: MySQL 8.0+

##功能需求

    1.学术调研
        1 论文搜索
        2 思维导图
        3 快速总结
        4 一键加入知识库
        5 历史搜索
    2.知识库
        1.增加/删除/命名 知识库文件夹
        2.知识库下上传/删除/批量删除
        3.基于知识库询问
        4.知识库搜索
        5.知识库总结
    3.网页搜索
        1.基于抖音/小红书/网页搜索-返回md格式的回答
        2.保存
        3.历史搜索
    4. ai服务
        1.工具列表(外部网页)
        2.提示词优化
        3.Bio生物医学助手
    5. 研究计划
        1.番茄钟
        2.定时/计时
        3.工作计划list
        4.日历
        5.打卡
    6.放松一下
        1.看你八字
        2.今天吃什么？
    7.个人主页
        1.编辑资料
        2.关联微信/飞书
